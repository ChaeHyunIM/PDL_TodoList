# TodoList
`Recoil`과 `React-beautiful-dnd` 라이브러리를 활용한 TodoList 과제입니다.

## 시안

## 기능
* Add1 버튼 클릭 시 중요한 Task 생성
* Add2 버튼 클릭 시 덜 중요한 Task 생성
* 각 Task에 대한 정보 표시 (recoil selector 활용)
* 각 Task에 대하여 Drag and Drop 가능
